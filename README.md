# Cara Mengatasi (Instalasi Pillow)
1. Buka Terminal: Jika anda menggunakan VS Code, anda bisa langsung membuka terminal di bagian bawah VS Code (Ctrl + ~).
2. Jalankan Perintah Instalasi: Ketik perintah berikut dan tekan Enter: pip install Pillow
3. Tunggu Proses Selesai: Pastikan laptop Anda terhubung ke internet. Terminal akan mengunduh library tersebut hingga muncul pesan Successfully installed Pillow-....
4. Jalankan Ulang Kode Anda: Setelah berhasil terinstal, Anda bisa langsung menjalankan kembali file import tkinter as tk.py tersebut.
